## Be Clear and Direct

Say exactly what you want. Ambiguous instructions produce ambiguous results. Claude responds best to explicit, specific requests that leave no room for interpretation.

**Incorrect (what's wrong):**

```text
Help me with this code
```

**Correct (what's right):**

```text
Review this Python function for bugs. Focus on edge cases in the error handling at lines 15-30.
```
