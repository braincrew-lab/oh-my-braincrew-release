## Define the Output Audience

Specify who will read the output. A summary for a junior developer differs fundamentally from one for a CTO. Audience definition shapes vocabulary, depth, and assumed knowledge.

**Incorrect (what's wrong):**

```text
Explain how this authentication flow works.
```

**Correct (what's right):**

```text
Explain how this OAuth2 flow works for a junior developer who has never implemented authentication. Avoid jargon; define any security terms you use.
```
